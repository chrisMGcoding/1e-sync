Mot clé Git
  • git init : creer une boite vide et invisible pour stocker les element(le faire une fois par projet)
  • git add --all : apres un travail, permet de faire une sauvegarde
  • git commit -m"contenu de l’étiquette" : permet de mettre une etiquette sur les sauvegardes
    • (git add --all et git commit -m vont toujours l’un apres l’autre
  • git log : permet de voir les element enregistrer par ordre (du plus recent au plus ancien)
  • les 3 suivante vont ensemble
    • git remote add origin https://github.com/agim-coroli/Gittest.git : permet de faire le lien entre votre machine et le site Gitub
    • git branch -M main : permet de créer, lister, renommer et supprimer des branches (?pas encors vu?)
    • git push -u origin main : : pousse les éléments enregistrer sur votre profil Gitub
